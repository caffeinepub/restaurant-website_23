# Specification

## Summary
**Goal:** Create a restaurant website with home, menu, about, and contact pages featuring editable pricing and contact information.

**Planned changes:**
- Create home page with welcome message, tagline, description, service highlights, and call-to-action
- Create menu page with three sections (Main Dishes, Fast Food, Drinks) displaying items with prices
- Create About Us page describing the restaurant's mission and values
- Create contact page with address, phone, hours, and email information
- Implement navigation between all four pages
- Add admin capability to edit menu prices and contact information
- Store editable data in Motoko backend
- Design warm, family-friendly visual theme with amber/terracotta/cream color palette

**User-visible outcome:** Users can browse a restaurant website with home, menu, about, and contact pages. Admins can edit menu prices and contact information, which persists across sessions.
